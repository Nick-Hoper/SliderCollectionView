# SliderCollectionView

>Swift4.2编写的分段选择器

![test.jpg](https://github.com/Nick-Hoper/SliderCollectionView/blob/master/test.jpg)


## Features

- 完美支持Swift4.2编译
- 采用collectionView封装
- 支持手势滑动
- 集成使用简单，二次开发扩展强大


## Requirements

- iOS 9+
- Xcode 9+
- Swift 4.0+
- iPhone

更详细集成方法，根据实际的例子请查看源代码中的demo
